# SPECTRA Presentation Diagrams

- Format: `SVG`
- Canvas: `1920x1080` (`16:9`, PowerPoint slide size)
- Purpose: import directly into `Figma`, then edit text, spacing, colors, or export for PPT

## Figma workflow

1. Create a new Figma frame at `1920 x 1080`.
2. Drag the SVG asset into the frame.
3. Ungroup only if you need to edit individual shapes.
4. Keep a `64 px` outer margin when placing titles or notes on the final slide.

## Files

- `01_use_case_diagram.svg` — slide-ready use case diagram for the internal SPECTRA platform
- `01_use_case_diagram_figjam.svg` — FigJam-style variant with simpler sticky-note layout for whiteboard use
- `02_system_architecture_figjam.svg` — FigJam-style high-level architecture diagram for the internal SPECTRA platform

## Style notes

- Palette follows the SPECTRA app theme: navy, gold, slate, blue, amber, green
- Text is sized for presentation readability first, not document-density
- Layouts are simplified so they remain understandable in PowerPoint
- The `*_figjam.svg` variants use softer fills, dashed containers, and sticky-note grouping so they feel natural inside FigJam
